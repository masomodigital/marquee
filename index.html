Learning HTML: Understanding the Marquee Tag (<marquee>)

Introduction

Welcome to the lesson on the <marquee> tag! The marquee tag is used to create scrolling text or images on a webpage.

Explanation

The <marquee> tag allows text to move across the screen in a specified direction, such as left, right, up, or down.

Example

Letâ€™s see an example in code:

<!DOCTYPE html>
<html>
<head>
    <title>My Marquee</title>
</head>
<body>
    <marquee>Welcome to my website!</marquee>
</body>
</html>

When you run this code, you'll see "Welcome to my website!" scrolling across the page.
